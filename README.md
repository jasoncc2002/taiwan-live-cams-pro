# Taiwan Live Cams Pro

A professional dashboard for exploring high-quality live camera streams from across Taiwan. Built with Next.js, Tailwind CSS, and shadcn/ui.

## Features

- **Real-time Live Streams**: Explore official scenic and city live cams from various regions of Taiwan.
- **Categorized View**: Easily switch between "City" and "Nature" views.
- **Dedicated Information Pages**: Each stream features detailed location descriptions and travel highlights.
- **Modern UI/UX**: Professional dark mode design with glassmorphism and fully responsive layout.
- **Fast Performance**: Powered by Next.js and optimized for speed.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Icons**: Lucide React
- **Hosting**: [Vercel](https://vercel.com/)

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Run the development server: `npm run dev`.
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy easily to Vercel using their official integration.
